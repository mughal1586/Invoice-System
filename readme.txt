C++ Invoice System

Description
This C++ program manages invoice data. It defines an `Invoice` class to store invoice details and calculates the invoice amount.

Features
-   Defines an `Invoice` class with attributes for part number, part description, quantity, and price per item.
-   Allows setting and getting invoice details.
-   Calculates the total invoice amount.
-   Provides functionality to insert and display invoice data.

Code Structure
The code consists of the following files:
-   `header.h`:  Contains the declaration of the `Invoice` class.
-   `Invoice System.cpp`: Contains the implementation of the `Invoice` class methods.
-   `main.cpp`: Contains the main function that uses the `Invoice` class to create invoice objects, set their data, and display their information.

Dependencies
-   iostream: For input/output operations.
-   string: For using string data type.

Compilation and Execution
To compile and run the program, save the files with their respective .cpp and .h extensions (e.g., header.h, Invoice System.cpp, main.cpp) and use a C++ compiler.
